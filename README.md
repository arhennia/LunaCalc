# 🐱 LunaCalc

> "A little black pixel cat helping you calculate at night."

LunaCalc is a cozy, minimal, and aesthetic desktop calculator companion built with Electron. She’s not just a tool; she’s a friend who lives on your screen, blinks at you, and occasionally meows.

![LunaCalc Preview](./src/renderer/assets/preview.png)

## ✨ Features

- **Alive & Expressive**: Luna has a personality state machine. She gets happy when you find answers, curious when you type, and recoils when you try to divide by zero.
- **Frameless & Floating**: A compact 200x280px window that stays always-on-top.
- **Haptic Sound Design**: Soft, synthesized pixel-chimes and a "meow" when you pet her (click her head!).
- **Keyboard First**: Full support for physical keyboard input with visual feedback on the buttons.
- **Retro Aesthetic**: Warm monochrome palette, 'Syne' & 'DM Mono' typography, and a subtle CRT scanline overlay.

## 🖱️ How to Interact

- **Drag**: Click and drag Luna's head to move her around your desk.
- **Pet**: Click her head to hear a cute meow! 🐾
- **Copy**: Double-click the result display to copy the value to your clipboard.
- **Keyboard**:
  - `0-9` for numbers
  - `+`, `-`, `*`, `/` for operations
  - `Enter` or `=` for results
  - `Backspace` to delete a digit
  - `Esc` to clear everything (AC)

## 🛠️ Installation

1. Clone this repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the app:
   ```bash
   npm start
   ```
